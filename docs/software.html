Software

Examples were calculated in R version 4.1.0 (2021-05-18) with an effort to use the most recent versions of R packages.

The R packages used in this book include (in alphabetic order):

  • fma (Hyndman 2020)
  • forecast (Hyndman et al. 2022)
  • ggplot2 (Wickham et al. 2022)
  • knitr (Xie 2022)
  • patchwork (Pedersen 2020)
  • pracma (Borchers 2021)
Borchers HW (2021) Pracma: Practical numerical math functions. R package version 2.3.6, https://CRAN.R-project.org/package=pracma
Hyndman R (2020) Fma: Data sets from "forecasting: Methods and applications" by makridakis, wheelwright & hyndman (1998). R package version 2.4, https://CRAN.R-project.org/package=fma
Hyndman R, Athanasopoulos G, Bergmeir C, et al (2022) Forecast: Forecasting functions for time series and linear models. R package version 8.16, https://CRAN.R-project.org/package=forecast
Pedersen TL (2020) Patchwork: The composer of plots. R package version 1.1.1, https://CRAN.R-project.org/package=patchwork
Wickham H, Chang W, Henry L, et al (2022) ggplot2: Create elegant data visualisations using the grammar of graphics. R package version 3.3.6, https://CRAN.R-project.org/package=ggplot2
Xie Y (2022) Knitr: A general-purpose package for dynamic report generation in r. R package version 1.38, https://yihui.org/knitr/
